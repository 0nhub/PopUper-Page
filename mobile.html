Frequently Asked Questions

Popuper is a small web tool to convert any website in a popup window, like an application.

Add a URL or type a link to the box and type ENTER to launch it as a popup window. You can also use our pre selection links below.

Be a minimalist. Focus just on one activity. Popuper helps you to increase your productivity by focusing just on one simple popup window without tabs or browser extensions.

Popuper is an open source project. The full project is for free. Popuper is created by volunteers, if you want to contribute, share popuper.org with your friends and family.
